Catalyst で Atom API テストツール #4

無事 XML::LibXML の make が通った（パッチあてた）ので、Atom API テストツール 「A2T2」のデモを公開します。好きに使っちゃってください。

それにともなって、見た目をほんのちょっとだけいじってみました。背景画像に gorouさん作 Background Image Maker で作った画像を使わせて頂いたり、アクセス中のインジケータとして、かぜぶろさんのとこで知った SanBaldo » Blog Archive » Ajax loading animated gif から GIF アニメ画像を頂いたり。感謝感謝。

デモといっても、テストツールとしてガンガン使ってくださって構いません。ご要望などありましたら、コメントください。
