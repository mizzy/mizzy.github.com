TWAN tag plugin for Jekyll

21 October 2011

I've written TWAN tag plugin for Jekyll which shows the image of The World At Night .

If you call this plugin in a post like this: 

{% raw %}{% twan 3003406 %}{% endraw %}

The image shows up like this.
